# vue-cli 安裝
> 1. 全域方式安裝 vue-cli
>	 -  `npm install -g @vue/cli`
> 2. 驗證 vue 安裝版本
> 	 - `vue -V`
> 3. 建立 vue 3 專案
> 	 - `vue create blog`
> 4. 進入專案
> 	 - `cd blog`
---
# Quasar 安裝
> 1. 全域方式安裝 Quasar CLI
>	 - `npm install -g @quasar/cli`
> 2. 驗證 Quasar 安裝版本
>	 - `quasar -v`
> 3. 部屬 Quasar 環境
>	 - `npm init quasar`
> 4. 進入專案
> 	 - `cd blog`
> 5. 開始開發
>	 - `quasar dev`
